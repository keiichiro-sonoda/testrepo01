# クロスワード自動解答ツール

> **「画像を渡せば解いて正解検証まで」の手順書は [SOLVING_GUIDE.md](SOLVING_GUIDE.md) にまとめています。**
> 半自動フロー（黒マス検出 `detect_grid.py` → 盤面モデル `grid.py` → 完成図 `render_solution.py`）と、
> 今回ハマった難所（薄い黒マス・小書き仮名の大書き・交差での訂正・ボーナスの意味検算）を記載。
> 下記の `solve.py` は「写真1枚をそのまま API に投げる」別アプローチです。

---


新聞などのクロスワードパズルを**撮影した写真を1枚渡すと、全マスを自動で埋めて返す**ツールです。
信濃毎日新聞の「土曜クロスワード」のような、日本語（カタカナ）のクロスワードを想定しています。

## なぜAI（Claude vision）に丸ごと任せるのか

クロスワードを写真から自動で解くには、大きく2つの壁があります。

1. **盤面・番号・カギ文章の読み取り**（画像認識／OCR）
2. **カギの答えを当てる**（例:「藤原○○○＝平安時代の政治家」「日本で最も長く続いた元号」）

特に 2 は、広い一般常識・時事・固有名詞の知識と推論が必要で、従来型OCR＋辞書だけでは解けません。
そこで本ツールは、**画像も読めてクイズも解けるAI（ClaudeのvisionモデルAPI）に画像を丸ごと渡し、
読み取り → 各カギの解答 → タテヨコ交差の整合（制約充足）→ 全マス穴埋め までを一括で行わせます**。
「写真1枚 → 完成盤面」を1回のAI呼び出しで実現する、いま最も現実的で精度の出る構成です。

## セットアップ

```bash
pip install -r requirements.txt
export ANTHROPIC_API_KEY=sk-ant-...   # Anthropic の APIキーが必要です
```

APIキーは <https://console.anthropic.com/> で発行できます（従量課金）。

## 使い方

```bash
# 基本（写真を渡すだけ）
python3 solve.py photo.jpg

# 構造化結果(JSON)も保存
python3 solve.py photo.jpg --json answer.json

# 安いモデルで試す（精度は少し落ちます）
python3 solve.py photo.jpg --model claude-sonnet-4-6

# 文字が小さいときは複数枚（盤面の接写＋カギの接写）を渡すと精度が上がる
python3 solve.py grid_closeup.jpg across_clues.jpg down_clues.jpg
```

## 撮影のコツ（精度を上げるために）

- 盤面とカギ文章の**両方**が1枚に入るように、**真上から・明るく・ピントを合わせて**撮る
- 影や折り目、指の写り込みを避ける
- 文字が小さいときは、盤面とカギを**別々に接写して複数枚**渡してください
  （`python3 solve.py 盤面.jpg ヨコ.jpg タテ.jpg` のように複数指定できます）。
  実際に手元の写真で試したところ、**盤面の黒マス・番号・交差の把握**が最大の難所で、
  全体を1枚で撮るより接写を分けたほうが安定します

## 仕組み / カスタマイズ

- `solve.py` … 画像をbase64化して Claude に送り、決められたJSON形式で解答を受け取り、
  盤面とカギ一覧を整形表示します。プロンプトは `SYSTEM_PROMPT` に集約しているので、
  別の新聞・パズル形式に合わせて調整できます。
- 既定モデルは `claude-opus-4-8`（最も賢い）。`--model` で変更可。

## 制限・注意

- カギは知識クイズなので、固有名詞や最新の時事は**間違える場合があります**。
  `confidence` と `notes` を確認し、最終的には人が確認してください。
- 写真が不鮮明だと読み取り精度が落ちます。撮影のコツを参照。
- 応募などに使う場合、**自動解答の正しさは保証されません**。あくまで補助ツールです。
